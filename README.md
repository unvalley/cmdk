# @command-palette

Library-agnostic command palette primitives.

## Packages

- [`@command-palette/react`](./packages/react/README.md)
  Unstyled React components for building command menus.
- [`@command-palette/vue`](./packages/vue/README.md)
  Unstyled Vue components for building command menus.
- [`packages/core`](./packages/core/README.md)
  Internal store and filtering engine used by adapters.

## Install

```bash
# React
pnpm add @command-palette/react

# Vue
pnpm add @command-palette/vue
```

Other libraries are incoming.
